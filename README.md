# Missing-person-registry

The Missing Persons Registry is a web-based application designed to facilitate the reporting, tracking, and dissemination of information about missing persons. Built with Django (Python web framework), the system provides a centralized platform where concerned individuals can report missing persons, view active cases, report sightings, and share information across social media platforms to increase awareness and aid in locating missing individuals. Project Type: Web Application Technology Stack: Django 4.2+, Python 3.x, SQLite, Bootstrap 5, HTML5 Target Users: General public, families of missing persons, law enforcement support

How to use:

Install Django using pip install django.
Create a project Django--admin startproject . Jump to your project directory using cd command.
Create a application Python manage.py startapp .
Now copy the content of project into your project folder and the same with application (replace the files if you want).
Make migrations Python manage.py makemigrations.
Finalize migrations Python manage.py migrate.
Start the development server Python manage.py runserver
